The Avatar component is a versatile and compact element used for representing
users or entities via images or initials. It's commonly used in headers, user
interfaces, or anywhere you need to display a user's picture or initials.

## Usage

Start by importing the necessary components from the package:

```tsx
import { Avatar, AvatarFallback, AvatarImage } from '@spren-ui/components/avatar';
```
